1.初めてのGIT

1.マスターリポジトリにカレントを変更する。

2.マスターリポジトリからローカルリポジトリにダウンロードする。

コマンドは以下の通り。
 git pull

3.ローカルファイルを編集する。

4.編集したファイルとローカルリポジトリのファイルの差分を確認する。

 git diff

5.編集したファイルをローカルリポジトリに反映する。

 git add <ファイル名>

6.ローカルリポジトリに反映する。

 git commit

7.マスターリポジトリにローカルリポジトリの内容を反映する。

git push

7.1.pushに失敗した場合

git fetch
git rebase origin/<マスターリポジトリ>

7.2.rebaseに失敗したとき

git rebase --continue

2.ブランチ

2.1ローカルリポジトリに新しいブランチを作成する

git checkout -b <ブランチ名>

2.2.ブランチをマスタリポジトリにアップロードする

git push origin <ブランチ名>
