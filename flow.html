1.初めてのGIT

1.1.セントラルリポジトリにカレントを変更する。

1.2.セントラルリポジトリからローカルリポジトリにダウンロードする。

コマンドは以下の通り。
 git pull

1.3.ローカルファイルを編集する。

1.4.編集したファイルとローカルリポジトリのファイルの差分を確認する。

 git diff

1.5.編集したファイルをローカルリポジトリに反映する。

 git add <ファイル名>

1.6.ローカルリポジトリに反映する。

 git commit

1.7.セントラルリポジトリにローカルリポジトリの内容を反映する。

git push

1.7.1.pushに失敗した場合

git fetch
git rebase origin/<セントラルリポジトリ名>

1.7.2.rebaseに失敗したとき

git rebase --continue

2.ブランチ

2.1.ローカルリポジトリに新しいブランチを作成する

git checkout -b <ブランチ名>

2.2.編集したファイルをローカルリポジトリのブランチに反映する

git add <ファイル名>
git commit

2.3.ブランチをセントラルリポジトリにマージする

git fetch
git rebase origin/<セントラルリポジトリ>

2.4.ブランチをセントラルリポジトリにアップロードする

git checkout <セントラルリポジトリ名>
git reset --hard <ブランチ名>
git push
